Поздравляем наш сайт Akkord-gitar с получением PR (PageRank) "1"

      Сегодня 03.04.2010 состоялся долгожданный ап PR. Для некоторых он прошел успешно, в том числе и для нашего сайта Akkord-gitar. Все долго с нетерпением ждали этого апа. И теперь, разумеется, все точно так же с нетерпением будут ждать следующего апа!)
     Многие начинающие seo оптимизаторы сталкиваются с двумя задачами. Это получение ТИЦ (Yandex) и PR (Google).

     ТИЦ (Yandex) - это технология поисковой машины «Яндекс», заключающаяся в определении авторитетности интернет-ресурсов с учётом качественной характеристики — ссылок на них с других сайтов. тИЦ рассчитывается по специально разработанному алгоритму, в котором особое значение придаётся тематической близости ресурса и ссылающихся на него сайтов. Данный показатель в первую очередь используется для определения порядка расположения ресурсов в рубриках каталога «Яндекса». При этом на соответствующих страницах каталога указываются лишь округлённые значения, которые помогают приблизительно ориентироваться в авторитетности ресурсов раздела.

     PR (PageRank Google) - один из алгоритмов ссылочного ранжирования. Алгоритм применяется к коллекции документов, связанных гиперссылками, и назначает каждому из них некоторое численное значение, измеряющее его «важность» или «авторитетность» среди остальных документов. Вообще говоря, алгоритм может применяться не только к веб-страницам, но и к любому набору объектов, связанных между собой взаимными ссылками, то есть к любому графу.


      Но сегодня я вам расскажу именно про PageRank Google так как ап PR состоялся именно сегодня.

      Как раccчитывается PageRank?
      PageRank или по-другому PR это что-то вроде оценки вашего сайта в Google. Так же PR - это алгоритм расчета этой оценки. Чем больше PR, тем выше сайт в результатах поиска. Например сайт с PR "0" или "-" это сайт новый, либо забаненный Гуглом. Сам Google имеет показатель PR "8". Всего максимальное значение PR "10". Показатель "10" имеют исключительно популярные в мире проекты.

      PageRank сайта становится больше, чем больше ссылок со страниц с PageRank на него ссылаются. Плюс немало важно качество этих ссылок. Например, одна ссылка с PR "5" гораздо более значимая, чем десять ссылок с PR "1".

      Ссылающаяся страница передает часть своего PR. Но сама она PR не лишается. Чем больше ссылок на разные сайты (или страницы), тем меньше доля PR достается каждой ссылке.

      Например, если страница с PR "3" ссылается на пять разных сайтов, то каждый из них получает больше PR, чем каждый из ста сайтов на которые ссылается страница с PR "8". Далее не имеют смысла ссылки со страниц, например с PR "1" на странице с PR "8", для этой страницы будет полезна ссылка с PR "9". Не имеет значения количество ссылок с одной и той же страницы, они воспринимаются Гуглом как одна ссылка. Например, если с сайта "Х" ведут три ссылки с главной страницы на сайт "Y" и одна на сайт "Z", то страницы "Y" и "Z" получают одинаковый PR.

      Мнение специалистов склоняются к тому, что в исчислении PR Гуглом применяется логарифмическая шкала. Это означает, что поднять PR сайта с "1" до "2" в разы проще, чем, например, с "4" до "5".
